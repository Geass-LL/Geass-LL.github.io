Crate是一个独立的可编译单元，它编译后会生成一个可执行文件或者一个库。

Package 就是一个项目，因此它包含有独立的 Cargo.toml 文件，以及因为功能性被组织在一起的一个或多个包。一个 Package 只能包含一个库(library)类型的包，但是可以包含多个二进制可执行类型的包。

Module 允许嵌套，能控制可见性。

多个Module组成Crate，多个Crate组成Package，进一步可以组成WorkSpace。
